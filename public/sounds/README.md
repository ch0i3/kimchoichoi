# 음원 넣는 곳

이 폴더에 파일을 넣으면 `/sounds/파일명` 으로 서빙됩니다 (Vite public 폴더).

| 파일 | 채널 | 용도 |
|---|---|---|
| `chamseon.mp3` | 참선 | 루프 배경음악 |
| `pohaeng.mp3` | 포행 | 루프 배경음악 |
| `beonnoe.mp3` | 번뇌 | 루프 배경음악 (잔잔하게) |
| `dadam.mp3` | 다담 | 채팅 보낼 때 **한 번** 재생되는 물소리(원샷, 루프 아님) |

동작
- 채널을 넘어가면 **이전 채널 음악은 즉시 끊기고**, 새 채널 배경음악만 잔잔히 페이드인됩니다.
- 다담은 배경 루프가 없고, 채팅을 보낼 때마다 물소리가 한 번 납니다.
- 우상단 버튼으로 음소거할 수 있습니다.
- 파일이 없는 채널은 조용히 무음 처리됩니다.
- 형식은 mp3 권장(호환성). 파일을 넣으면 코드 수정 없이 바로 반영됩니다.
